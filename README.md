## 📌 Descripción
Repositorio destinado a la práctica y desarrollo de aplicaciones web utilizando el framework **Django**. Este proyecto sirve como base para explorar la arquitectura MTV (Model-Template-View), la gestión de bases de datos mediante el ORM de Django, el sistema de enrutamiento de URLs y la administración integrada.

## 🚀 Características y Aprendizajes
* Configuración inicial y estructuración de un proyecto web escalable en Django.
* Modelado relacional de bases de datos haciendo uso del ORM.
* Creación de vistas (basadas en funciones y en clases) interactuando con el sistema de *Templates*.
* Implementación y personalización del Panel de Administración (`/admin`).
* Gestión de archivos estáticos y rutas dinámicas.

## 🛠️ Stack Tecnológico
* **Backend:** Python 3.x, Django
* **Base de Datos:** SQLite (Entorno de desarrollo)
* **Frontend:** HTML5, CSS3, plantillas nativas de Django (Jinja2)
